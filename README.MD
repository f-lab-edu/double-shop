## 프로젝트 목표

- coupang, 마켓컬리와 같은 이커머스 기반 서비스를 개발하는 것이 목표인 프로젝트 입니다.
- 애플리케이션 프로토타입은 카카오 오븐을 사용하였습니다.
- 기능 개발에 더불어, 코드의 재사용성 및 유지보수 효율성 확보하는 것을 목표로 개발을 하고 있습니다.
- CI/CD를 통한 자동화를 구현하여, 협업에 용이한 프로젝트를 구성하였습니다.

## 사용한 기술 & 개발 환경

- JAVA 11
- SpringBoot
- SpringSecurity
- Maven
- MySQL
- MyBatis
- Redis
